# Named colors

The names of available colors are stored in alphabetical order in the dictionary `Colors.color_names`:

```
color_names = Dict(
    "aliceblue"            => (240, 248, 255),
    "antiquewhite"         => (250, 235, 215),
    "antiquewhite1"        => (255, 239, 219),
    ...
```
